I sometimes get asked what my process is and what is included during each step. Generally I follow this process for every project.

  1. Discovery
    • Learning business objectives
    • Prioritize features
    • View past statistics
    • User Research
    • User Stories and Personas
    • Brainstorming session
  2. Planning
    • Information architecture
    • Build a site-map
    • Discuss navigation
    • Project milestones
  3. Interactive Prototype
    • Basic sketches
    • Design overall look and feel
    • Integrated color, typography, images, & copy
    • Frontend Development (HTML, CSS, JS)
  4. Backend
    • Integrate CMS
  5. Testing
    • Deploy to beta
    • Usability
    • Load
    • Accessibility
    • Train users
  6. Deploy
    • Deploy to production
  7. Review
    • Establish a road-map for improvements
    • Analytics
  8. Manage
    • Execute improvements
